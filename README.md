# BLEIBGUT — Content-Assets (öffentlich)

Fertig gerenderte Social-Media-Medien (Karussell-Slides, Short-Videos).
Postiz holt sie von hier per raw-URL — dieses Repo ersetzt den früheren
Shopify-CDN-Umweg (aus der Cloud-Sandbox nicht erreichbar, Proxy 403).

URL-Muster:
https://raw.githubusercontent.com/BLEIBGUT/bleibgut-content-assets/main/<ordner>/<datei>

Quelldateien (slides.html, Captions) liegen im privaten Repo
`bleibgut-website` unter `content-pipeline/`.
